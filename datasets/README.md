# Processed Concrete Datasets

These CSV files were copied from the `Data-Processing` branch at `data/processed`.

## Files

- `combined_concrete.csv`: original combined normal concrete and UHPC dataset.
- `normal_concrete.csv`: original normal concrete subset.
- `uhpc_concrete.csv`: original UHPC subset.
- `update_combined_concrete.csv`: updated combined dataset with `steel_fiber` and `sand_binder_ratio`.
- `updated_normal_concrete.csv`: updated normal concrete subset.
- `update_uhpc_concrete.csv`: updated UHPC subset.

Use the updated normal and UHPC files for direct feature-importance comparisons because they have matching schemas and include the newer composition fields.
